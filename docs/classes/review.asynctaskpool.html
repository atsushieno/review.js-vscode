Class AsyncTaskPool

同期化処理を非同期化するためのヘルパクラス。 array.forEach(value => process(value)); を以下のように書き換えて使う。 var pool = new AsyncTaskPool(); array.forEach(value => pool.add(()=> process(value)); pool.consume().then(()=> ...);
