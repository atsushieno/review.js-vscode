Interface ITreeVisitor

構文木を渡り歩くためのVisitor。 実装されなかったメソッドは、visitDefault または NodeSyntaxTree を継承している場合 visitNode にフォールバックする。 各メソッドの返り値としてanyを返す。 undefined, true を返した時、子要素の探索は継続される。 false を返した時、子要素の探索は行われない。 Function を返した時、子要素の探索を行う代わりにその関数が実行される。Functionには引数として実行中のTreeVisitorが渡される。
